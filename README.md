#  Личный дневник питания (Food Diary)

##  О проекте

**Food Diary** — это удобное веб-приложение для ведения личного дневника питания.  
Проект помогает пользователям отслеживать потребляемые продукты, контролировать калорийность рациона и анализировать баланс БЖУ (белков, жиров и углеводов).

---

##  Сущности базы данных

### 1. Продукт
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | UUID | Уникальный идентификатор |
| `название` | String | Наименование продукта |
| `калории` | Float | Ккал на 100 г |
| `белки` | Float | Белки на 100 г |
| `жиры` | Float | Жиры на 100 г |
| `углеводы` | Float | Углеводы на 100 г |

### 2. Приём пищи
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | UUID | Уникальный идентификатор |
| `название` | String | Завтрак / Обед / Ужин / Перекус |
| `порядок` | Integer | Для сортировки (1-4) |

### 3. Запись
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | UUID | Уникальный идентификатор |
| `дата` | Date | Дата приёма пищи |
| `продукт_id` | UUID | Ссылка на продукт |
| `приём_пищи_id` | UUID | Ссылка на тип приёма |
| `вес` | Float | Вес порции в граммах |

##  Функциональные возможности

### Для пользователя 
- [x] Добавление записей о питании
- [x] Выбор продукта из базы
- [x] Указание веса порции (авторасчёт калорий и БЖУ)
- [x] Просмотр дневной статистики
- [ ] Графики потребления калорий по дням
- [ ] Установка личных норм калорий и БЖУ

### Для администратора 
- [x] Управление базой продуктов
- [x] Добавление новых продуктов
- [x] Редактирование существующих
- [x] Удаление устаревших записей

---

##  Графики и отчёты

Приложение предоставляет визуализацию данных:

-  **Линейный график** — динамика калорий за неделю/месяц
-  **Круговая диаграмма** — распределение БЖУ за день
-  **Столбчатая диаграмма** — сравнение приёмов пищи

---

##  Технологический стек

- **Frontend:** `React` / `Vue.js`
- **Backend:** `Node.js` + `Express`
- **База данных:** `PostgreSQL`
- **Графики:** `Chart.js` / `D3.js`

##  Запуск проекта

```bash
# Клонирование репозитория
git clone https://github.com/your-username/food-diary.git

# Переход в директорию
cd food-diary

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env

# Запуск миграций БД
npm run migrate

# Запуск приложения
npm run dev
```
