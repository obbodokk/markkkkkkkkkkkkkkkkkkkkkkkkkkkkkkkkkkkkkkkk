## 📋 О проекте «Цветик-Семицветик»

**Цветик-Семицветик** — это инновационное веб-приложение для загадывания желаний.  
Продукт разработан специально для инвесторов из Заморского Царства и сочетает в себе магию исполнения желаний с современными технологиями.

**Технологический стек:**
- **Frontend:** `HTML`, `CSS`, `JavaScript`
- **Backend:** `Node.js`
- **Инфраструктура:** Виртуальный хостинг «Тридевятое Царство & Три Топора» 🪓🪓🪓

---

## 🚀 Как запустить проект

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/tri-topora/cvetik-semicvetik
   ```
   или перейдите по [ссылке на репозиторий](https://github.com/tri-topora/cvetik-semicvetik)

2. **Перейдите в папку проекта:**
   ```bash
   cd cvetik-semicvetik
   ```

3. **Установите зависимости:**
   ```bash
   npm install
   ```

4. **Запустите сервер:**
   ```bash
   npm start
   ```

## ✨ Функциональные возможности

- [**Главная страница**](/#main) — приветствие и магия исполнения желаний
- [**Страница загадывания желаний**](/#wish) — интерактивная форма с семицветиком
- [**История желаний**](/#history) — архив всех загаданных желаний
